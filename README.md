# hello-world

This repository contains an Apple Watch breathwork application.

## BreathworkWatch App

A comprehensive Apple Watch app for guided breathing exercises, featuring:

- **15 breath patterns** including meditation and kundalini techniques
- **Visual animations** with color-coded breath phases
- **Haptic feedback** to guide inhales, exhales, and holds
- **Multiple categories**: Meditation, Kundalini, Calming, and Energizing patterns

### Features

- Box Breathing, 4-7-8 Breathing, Coherent Breathing
- Kundalini techniques: Breath of Fire, Long Deep Breathing, Alternate Nostril
- Real-time visual feedback with expanding/contracting animations
- Tactile haptic guidance for each breath phase
- Clean, minimal watchOS interface

See [BreathworkWatch/README.md](BreathworkWatch/README.md) for complete documentation.
